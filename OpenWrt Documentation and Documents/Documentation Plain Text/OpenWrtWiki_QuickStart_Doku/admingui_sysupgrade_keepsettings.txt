====== Веб-графический интерфейс Sysupgrade: флажок "Сохранить настройки" ======
Флажок "Сохранить настройки" является более сложной функцией. \\
Если вы не точно понимаете варианты использования кнопок, **снимите флажок Держать настройки"** каждый раз, когда вы запустили новый системный экран LEDE на свое устройство, чтобы **не** сохранять настройки.


    * Только установите флажок «Сохранить настройки» на небольших светодиодных индикаторах LEDE → LEDE, которые, как известно, не меняют конфигурационную структуру 
    * Используйте его только для одного и того же канала прошивки (выпуск → выпуск, моментальный снимок → моментальный снимок)
    * При проверке он сохранит несколько определенных файлов конфигурации при обновлении, но не весь оверлейный раздел. 
    * Если вы регулярно запускаете свое устройство, рекомендуется сразу же снять флажок «Keep Settings» при каждом запуске маршрутизатора и вместо этого создать собственный сценарий установки для вашей настройки. Пример: скрипт [[https://github.com/richb-hanover/OpenWrtScripts/blob/master/config-openwrt.sh|config-openwrt.sh]].
    * "Сохранять настройки" также можно использовать для сохранения собственных файлов конфигураций. Для этого перейдите в  **System > Backup/Flash Firmware** и откройте вкладку **Configuration** добавьте конфигурацию и нажмите ”Submit”. Это позволит вам добавить ваши настраиваемые файлы конфигурации в''/etc/sysupgrade.conf'' файл, который считывается sysupgrade при резервном копировании файлов. Нажмите "Submit" после завершения редактирования.\\ Чтобы просмотреть все файлы, которые будут храниться, когда установлен флажок «сохранить настройки», нажмите кнопку  "**Open list...**".
