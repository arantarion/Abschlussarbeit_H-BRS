====== PPTP ======
<WRAP important>
PPTP is considered insecure due to [[wp>Point-to-Point_Tunneling_Protocol#Security|multiple vulnerabilities]].
Use another VPN solution if possible.
</WRAP>

The [[wp>Point-to-Point_Tunneling_Protocol|PPTP]] is a method for implementing VPN.
PPTP uses a control channel over TCP and a GRE tunnel operating to encapsulate PPP packets.
pptpd is server daemon which enables pptp clients to establish tunnel over IP network.
