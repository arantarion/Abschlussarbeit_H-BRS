====== 使用Docker创建OpenWrt固件 ======

参阅: [[docs:guide-user:virtualization:docker_openwrt_image|Docker OpenWrt Image]]

你可以使用预制的[[https://www.docker.io/|Docker]]镜像来编译你自己的OpenWrt固件。这种Docker镜像的列表如下：（你可以在这些项目的首页或README文件找到使用方法）：

  * [[https://github.com/noonien/docker-openwrt-buildroot|noonien/docker-openwrt-buildroot]]
  * //wlan slovenija// [[https://github.com/wlanslovenija/firmware-core|image generator]]

===== 基于云平台的OpenWrt固件编译环境API =====

为了便于分享，重用，交换OpenWrt固件编译环境，同时便于测试社区中的新固件。[[https://github.com/wlanslovenija|wlan slovenija]]推荐了一种云平台编译环境API。
详细说明可以在以下连接中找到。
[[https://github.com/wlanslovenija/firmware-core#openwrt-cloud-builder-api|建议采用标准]].
