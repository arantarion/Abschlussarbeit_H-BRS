====== User Configuration ======
This UCI configuration file serves as a central user database for different services. So you do not have to create users in /etc/passwd that would be able to logon to the OpenWrt device. At the moment only racoon is supported.

===== Sections =====
==== user ====
^Name^Type^Required^Default^Description^
|enabled|boolean|no|0|Active flag. User is available for authentication (=1) or not (=0)|
|name|string|yes|(none)|Name of the user|
|password|string|yes|(none)|Password of the user|
|xauth|boolean|no|0|User is available for IPsec [[docs:guide-user:services:vpn:strongswan:roadwarrior|road warrior]] XAuth hybrid authentication if set to 1|
|crt_subject|string|no|(none)|Subject of the user/machine certificate for [[docs:guide-user:services:vpn:ipsec:start|road warriors]]|

===== Examples =====
<code bash>
config 'user'
  option 'enabled' '1'
  option 'name' 'otto'
  option 'password' 'this_is_ottos_password'
  option 'xauth' '1'
  option 'crt_subject' 'CN=otto01@acme.com'
</code>
