====== USB 3.0和WiFi的问题======
不幸的是，2.4 GHz的USB 3.0和无线网络会干扰相同的频谱，这通常会导致问题。

详见[[http://uk.pcmag.com/networking-reviews-ratings-comparisons/13179/opinion/wireless-witch-the-truth-about-usb-30-and-wi-fi-interference|PC Mag Article关于USB3.0和2.4 GHz无线问题]]。

这些干扰造成的问题不会影响驱动器活动，但是如果在您的LEDE wifi设备上连接了USB3.0存储设备，可能会严重影响2.4 WiFi。

  * 2.4 WiFi客户端可能无法连接到2.4 GHz WiFi收音机

  * 已连接2.4 WiFi客户端遇到连接超时或严重失速或中断数据的传输

这些问题很可能也会在驱动器空闲时出现。

解决方法可能因您的情况而异，您可以尝试：

  * 使用5 GHz WiFi为您的WiFi客户端设备，因为这个问题只影响2.4 WiFi。

  * 使用USB2.0驱动器外壳代替USB3.0，因为USB2.0不会干扰2.4 GHz频谱或禁用USB3驱动程序强制回退到USB2.0。目前许多LEDE设备的USB3.0数据传输速度与2.0相比并不明显。

  * 如果您的USB3.0存储设备是一个外部USB3.0硬盘驱动器，您可以尝试不同的外壳，希望有更好的屏蔽。

  * 您可以尝试使用不同的USB3.0电缆，可能是USB3.0设备附带的供应商提供的电缆可能屏蔽不良。但不幸的是，几家USB外壳厂商在驱动器端使用带有专有USB连接器的电缆，这使得用户并不容易在市场上获得更好的屏蔽型号的设备。

  * 您可以尝试将您的USB3.0存储设备重新放置在远离LEDE WiFi设备的位置。

  * 如果您在技术上有天赋，您可以尝试为您的USB3.0存储设备设置自定义屏蔽。

  * 也有可能是连接到您的LEDE WiFi设备的电源线或网络电缆屏蔽不良。

  * 您可以尝试2.4 GHz无线电使用不同的和固定的频道稍微改善这个问题：尝试频道1，6和11，并尝试WiFi信道宽度20（如果你以前使用40）。
